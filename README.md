# 🐍 Python Projects Collection

##Make heart - This is a simple Python program that uses the turtle graphics module to draw a heart shape on the screen. It sets up a window with a pink background and uses a turtle object to draw and fill the heart with red color. The program uses functions from the turtle module, movement commands, angles, loops (implicitly through drawing steps), and styling features like colors and pen size to create the design.

##My Arena - This is a Python chatbot program called “Arena” that interacts with the user and provides multiple features like suggesting movies, telling jokes, showing date and time, playing quizzes, and giving tips. It uses the random module for selecting movies, jokes, facts, and suggestions, and the datetime module to display the current date and time.
The program is built using classes and objects, functions, loops, conditionals, lists, dictionaries, and user input to manage conversations and different features. It also includes simple emotion detection and interactive responses, making the chatbot more engaging and dynamic.

##BlackJam - This is a Python Blackjack game where the player competes against the computer. Cards are randomly selected using the random module, and the game follows basic Blackjack rules like aiming for a score close to 21. The program uses functions, loops, conditionals, lists, and user input to manage the game flow, calculate scores, and decide the winner. It also includes features like handling Blackjack (21), adjusting Ace values, and allowing the user to draw or pass cards.

##Print_lyrics - This is a Python program that displays song lyrics (a tribute to Zubeen Garg) in a creative animated style. It prints each line with a typing effect using the time module, adds random visual effects with the random module, and uses sys and io for proper text encoding. The program also applies colorful gradient text using ANSI escape codes, creating a visually appealing output. It uses functions, loops, lists, conditionals, and string formatting to control the animation and presentation of the lyrics.

##Number Guessing Game -This is a simple Python Number Guessing Game where the computer randomly selects a number between 1 and 100 using the random module. The player guesses the number with limited attempts based on the chosen difficulty level (easy or hard). The program uses functions, loops, conditionals, and user input to control the game flow and provide hints like “too high” or “too low.” The game continues until the player guesses correctly or runs out of attempts.
